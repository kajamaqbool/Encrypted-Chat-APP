# Frontend — Encrypted Chat App

React + Vite frontend for the Encrypted Chat App.

## Tech Stack
- **React 18** — UI library
- **Vite** — Build tool
- **Zustand** — State management
- **Socket.IO Client** — Real-time messaging
- **TailwindCSS + DaisyUI** — Styling
- **Axios** — API requests
- **React Router v6** — Routing

## Run locally

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173` — requires backend running at `http://localhost:5001`.

## Build

```bash
npm run build
```

Output goes to `/dist` — served by the Express backend in production.
